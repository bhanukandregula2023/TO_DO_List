To-Do Application 



Objective

The objective of this project is to develop a responsive To-Do application using HTML, CSS, and JavaScript. The application helps users organize and manage their daily tasks efficiently by allowing them to add, edit, search, complete, and delete tasks. 

**Technologies Used**
HTML5 – Used to create the structure of the web page.
CSS3 – Used to design a modern, responsive, and user-friendly interface.
JavaScript (ES6) – Used to implement the application's functionality and DOM manipulation.



**Features Implemented**
1. Add Task:  Users can enter a task in the input field and click the Add button to add it to the task list.
2. Display Tasks: All added tasks are displayed dynamically in a clean card layout.
3. Mark Task as Completed: Users can mark tasks as completed using a checkbox.
Completed tasks:
Display a strike-through text
Change to a light green background
Are counted in the completed task counter
4. Edit Task: Users can edit an existing task through a custom popup (modal). After saving, the updated task is displayed immediately.

5. Delete Task: Users can permanently remove tasks from the list using the Delete button.

6. Search Task: A search bar allows users to filter tasks by typing the task name. The results update instantly as the user types.

7. Task Counter
The application displays:
Total Tasks
Completed Tasks
Pending Tasks

The counters are updated automatically whenever tasks are added, edited, completed, or deleted.


9. Responsive Design
The application is fully responsive and works properly on:
Mobile devices
Tablets
Laptops/Desktop

CSS Flexbox and Media Queries are used to adjust the layout for different screen sizes.


**Working of the Application**
-User enters a task.
-The task is stored as a JavaScript object inside an array.
-JavaScript dynamically creates task cards and displays them on the screen.
-Whenever a task is added, edited, deleted, or completed, the task list and counters are automatically updated.



**Conclusion**
This project demonstrates the use of HTML, CSS, and JavaScript to build a responsive web application with dynamic functionality. It includes task management features such as adding, editing, deleting, searching, marking tasks as completed. The project also follows modern UI design principles, making it both functional and user-friendly.

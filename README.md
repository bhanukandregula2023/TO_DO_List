# 📝 To-Do App

A simple and responsive To-Do App built using **HTML, CSS, and JavaScript**. This project allows users to add and manage their daily tasks through a clean and user-friendly interface.

## 🚀 Features

* Add new tasks
* Display all added tasks
* Mark tasks as completed
* Delete tasks
* Responsive and clean user interface
* Instant task updates without reloading the page

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## ▶️ How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any modern web browser.
4. Start adding and managing your tasks.
